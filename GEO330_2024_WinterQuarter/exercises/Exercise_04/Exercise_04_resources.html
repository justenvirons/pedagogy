Introduction

This relatively brief exercise utilizes the suburban Cook County Health Atlas and Chicago Health Atlas indicators to evaluate select public health outcomes relative to transportation access and quality across Cook County. The exercise will make use of indicators derived from local surveys (i.e., Cook County Health Survey and the Healthy Chicago Survey) that are administered annually by the two local health departments. Both surveys are designed to collect neighborhood-level information from residents on such things as health status, risk behaviors, and access to health care to inform and monitor the health status of communities. Download the exercise #4 presentation template to organize data elements and summaries. Submit the completed presentation to the appropriate submission folder on D2L.

Step 5. Create novel county-wide geographic layer for mapping

Use the following code to create a novel geography (N=202) that combines city of Chicago community areas (N=77) and suburban Cook County municipalities (N=125 using 4 square mile threshold area) using GEOJSON data imported from the two health atlases. Figure 2 displays an interactive map of the new geography (i.e., “novel_geo_cook”), which includes both suburban Cook County municipalities (dark gray) and community areas (light gray).

# import City of Chicago community areas (N=77) GEOJSON and transform to local projection for spatial analysis (epsg:3435)

community_areas <- st_read("https://metopio.blob.core.windows.net/lalage/atlas/3/shapes/communityareas.topo.json", quiet=TRUE) %>% 
  st_as_sf() %>% 
  st_set_crs(4326) %>% 
  st_transform(3435) %>% 
  rename(geoid = id) %>% 
  mutate(geoid= str_replace(geoid,"1714000-",""),
         location = "Chicago, IL",
         type = "community area")

# import places within suburban Cook County (N=125) and transform to local projection for spatial analysis (epsg:3435)

places_cook_county <- st_read("https://metopio.blob.core.windows.net/lalage/atlas/6/shapes/SCC-place.topo.json", quiet=TRUE) %>% 
  st_as_sf() %>% 
  st_set_crs(4326) %>% 
  st_transform(3435) %>% 
  rename(geoid = id) %>% 
  mutate(sqmi = as.numeric(st_area(geometry)) * 0.0000003861,
         location = "suburban Cook County",
         type = "municipality") %>% 
  filter(sqmi >= 4) %>% 
  select(-sqmi)

# create a novel geography in Cook County which combines both municipalities in suburban Cook County and communities areas in Chicago

novel_geo_cook <- places_cook_county %>% 
  bind_rows(community_areas) %>% 
  st_as_sf() %>% 
  st_make_valid()

a_topic_list <- c("obesity","safety", "sidewalk", "violence", "transit", "health", "inactivity") # edit here

  a_topic_url <- paste0("https://cookcountyhealthatlas.org/api/v1/data/?format=json&layer=place&period=2022&population=&topic=",atopic_key)

  a_topic_df <- a_topic_json$results %>% 
    as.data.frame() %>% 
   select(estimate = v,

          geoid = g,
          layer = l) %>%

  assign(a_table_name, a_topic_df)
  
}

# download Chicago health survey indicators
for (i in 1:length(a_chicago_topic_keys)){

  atopic_key <- a_chicago_topic_keys[i]

Step 8. Examine statistical relationships between selected variables

Use the code below to create a bivariate correlation plot to explore the statistical relationships between the seven downloaded indicator variables. Pearson’s bivariate correlation (or Pearson’s r) quantifies the strength and direction of the linear relationship between each pair of variables. The coefficients ranges from -1 to +1 with values greater than 1 indicating positive linear relationships (i.e., as one variable increases, the other variable also increases) and values less than 1 indicating a negative linear relationship (i.e., as one variable decreases, the other variable also decreases). Figure 3 shows two plots based on the same correlation matrix: Plot 1 shows explicit correlation statistics and plot 2 shows graduated circles.

# subset indicator data for statistical analysis
indicators_all_long_sub <- indicators_all_long_geo %>%
  st_drop_geometry() %>% 
  drop_na() %>% 
  select(-starts_with("se"), -c(geoid,name,type, location,layer))
  
# calculate bivariate correlations between all numeric variables
variable_biv_correlations <- cor(indicators_all_long_sub, method="pearson")

# create correlation plot option 1 with correlation values
corrplot_1 <- corrplot(variable_biv_correlations, method="number", col=c("black","white"), order="hclust", bg="lightgrey", tl.col = "grey")

# create correlation plot option 2 with graduated circles
corrplot_2 <- corrplot(variable_biv_correlations, col=c("black","white"), order="hclust", bg="lightgrey", tl.col = "grey")

# display correlation plot option 1
corrplot_1

# display correlation plot option 2
corrplot_2

# Step 8. Examine statistical relationships between selected variables

Use the code below to create a bivariate correlation plot to explore the statistical relationships between the seven downloaded indicator variables. Pearson's bivariate correlation (or Pearson's r) quantifies the strength and direction of the linear relationship between each pair of variables. The coefficients ranges from -1 to +1 with values greater than 1 indicating positive linear relationships (i.e., as one variable increases, the other variable also increases) and values less than 1 indicating a negative linear relationship (i.e., as one variable decreases, the other variable also decreases). @fig-correlation_plots shows two plots based on the same correlation matrix: Plot 1 shows explicit correlation statistics and plot 2 shows graduated circles.

```{r}
#| warning: false
#| eval: false
#| echo: true
#| label: fig-bivariate_correlation_plots

# subset indicator data for statistical analysis
indicators_all_long_sub <- indicators_all_long_geo %>%
  st_drop_geometry() %>% 
  drop_na() %>% 
  select(-starts_with("se"), -c(geoid,name,type, location,layer))
  
# calculate bivariate correlations between all numeric variables
variable_biv_correlations <- cor(indicators_all_long_sub, method="pearson")

# create correlation plot option 1 with correlation values
corrplot_1 <- corrplot(variable_biv_correlations, method="number", col=c("black","white"), order="hclust", bg="lightgrey", tl.col = "grey")

# create correlation plot option 2 with graduated circles
corrplot_2 <- corrplot(variable_biv_correlations, col=c("black","white"), order="hclust", bg="lightgrey", tl.col = "grey")

# display correlation plot option 1
corrplot_1

# display correlation plot option 2
corrplot_2
```
